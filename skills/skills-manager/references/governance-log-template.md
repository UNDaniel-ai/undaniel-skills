# Governance Log Template

Use this template after each task.

```markdown
## Governance Record

- task_id: <task identifier or timestamp>
- touched_skills: <comma-separated list>
- candidate_found: <yes|no>
- decision: <update-existing|create-new|none>
- decision_reason: <boundary-based reason>
- confirmation_step_1: <user confirmation summary>
- confirmation_step_2: <user confirmation summary>
- changes_applied: <short list of files/behavior changed>
- validation: <commands executed + result>
- followups: <optional>
```
